Summary

Defines a passport cas strategy to authenticate requests using a cas server.

Constructor

CasStrategy

Defined in F:\projects\openveo-portal\app\server\passport\cas\CasStrategy.js:15

Syntax

CasStrategy

(
  • options
)

Summary

Parameters:

  • options Object

    The list of cas strategy options

    • service String

      The service to use to authenticate to the CAS server

    • version String

      The version of the CAS server

    • url String

      The url of the CAS server

    • certificate String

      The absolute path to the CAS server certificate

Example:

e.g. Configuration example
// {
//   "service": "https://openveo-portal.com", // Application service
//   "url": "https://openveo-cas.com:8443/cas", // CAS server url
//   "version": "3", // CAS protocol version (could be 1, 2, 3)
//   "certificate": "/home/test/cas.crt" // CAS certificate public key
// }
