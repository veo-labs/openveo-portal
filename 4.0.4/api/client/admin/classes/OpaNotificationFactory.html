Summary

Displays a notification using AngularJS Material toaster.

Parameters:

  • message String

    The message to display

  • [delay] Number optional

    Duration (in milliseconds) before automatically closing the notification, 0 to add a button to close the notification

  • [closeLabel] String optional

    Close button label

  • [closeAriaLabel] String optional

    Close button ARIA label
