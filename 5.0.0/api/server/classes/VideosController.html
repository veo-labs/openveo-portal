Summary

Searches OpenVeo Publish videos.

Parameters:

  • request Request

    ExpressJS HTTP Request

    • body Object

      Request's body

      • filter Object
        Web service endpoint filters
      • [filter.query] String optional
        Filter to get only videos which contain the query inside the title or description
      • [filter.dateStart] String optional
        Filter to get only videos after a particular date
      • [filter.dateEnd] String optional
        Filter to get only videos before a particular date
      • [filter.sortBy] String optional
        The name of the property to sort by (either "views" or "date")
      • [filter.sortOrder] String optional
        The order of the sort (either "asc" or "desc")
      • pagination Object
        Pagination to set limit and page
      • [pagination.limit] Object optional
        Maximum number of videos to retreive
      • [pagination.page] Object optional
        The number of the page to retrieve
      • [include] String | Array optional
        The list of fields to include from returned videos
      • [exclude] String | Array optional
        The list of fields to exclude from returned videos. Ignored if include is also specified.
  • response Response

    ExpressJS HTTP Response

  • next Function

    Function to defer execution to the next registered middleware
