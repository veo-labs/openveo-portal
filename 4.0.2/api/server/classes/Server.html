del

)
String

Summary

Removes cache entries.

Override NodeCache.del to implement wildcards to be able to remove all cache entries corresponding to a wildcard. Only wildcard at the end of the entry id is supported, it is not possible to add a wildcard at the beginning of the id.

Parameters:

  • keys Array | String

    The list of keys to delete

  • callback Function

    The function to call when its done with:

    • Error An error if something went wrong
    • Number The number of delete entries

Returns:

String:

The number of deleted entries

Example:

cache.del('entry-id-with-wildcard-*');

get

Provided by the storages module.

Defined in /home/lhawsa/Documents/projects/openveo/openveo-portal/app/server/Cache.js:65

Syntax

get

(
  • key
  • callback
  • errorOnMissing
)
String | Array | Undefined

Summary

Gets cache entry.

Override NodeCache.get to implement wildcards to be able to get all cache entries corresponding to a wildcard. Only wildcard at the end of the entry id is supported, it is not possible to add a wildcard at the beginning of the id.

Parameters:

  • key String

    The key to look for

  • callback Function

    The function to call when its done with:

    • Error An error if something went wrong
    • Mixed The value of the cache entry if no wildcard used or the values of the cache entries when a wildcard is used
  • errorOnMissing Boolean

    true to throw an error if no cache entry is not found

Returns:

String | Array | Undefined:

The value of the cache entry if no wildcard used or the values of the cache entries when a wildcard is used

Example:

cache.get('entry-id-with-wildcard-*');

Summary

Gets cache entries.

Override NodeCache.mget to implement wildcards to be able to get all cache entries corresponding to a wildcard. Only wildcard at the end of the entry id is supported, it is not possible to add a wildcard at the beginning of the id.

Parameters:

  • keys Array

    The list of cache entries to fetch

  • callback Function

    The function to call when its done with:

    • Error An error if something went wrong
    • Object The values of the cache entries with cache id as the key and cache value as the value

Returns:

Object:

The values of the cache entries with cache id as the key and cache value as the value

Example:

cache.mget(['entry-id-with-wildcard-*']);
